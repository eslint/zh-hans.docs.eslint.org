[eslint-config-eslint](https://www.npmjs.com/package/eslint-config-eslint) 定义了 ESLint 的代码约定。

可以通过查看任何给定规则的项目文档，找到使用特定规则的基本原理。如果是我们定义的规则，请参阅我们自己的[规则文档](https://eslint.org/docs/rules/)，否则，请参阅该规则的插件文档。
